== Installation ==
 
La instalación e muy simple con tan solo 3 pasos:
 
e.g.
 
1. Sube el archivo comprimido "wsme.zip" en la carpeta `/wp-content/plugins/` y descomprime
2. Activa el plugin en la opción 'Plugins' del menu de wordpress
3. Luego en la barra izquierda de tu menu de administración aparecerá la opción "WSME" donde podrás configurar tu número de whatsapp
